# References

Datasets:

* petdata_1000_100.csv - Random generated dataset (shape=(1000,100)). Code in `project/notebook/Petdata_generator.ipynb`. Rating codes:
    * NaN - Document not seen
    * 0 - Document seen, but not rated
    * 1 - 1 star (poor)
    * 2 - 2 stars
    * 3 - 3 stars (mediocre)
    * 4 - 4 stars
    * 5 - 5 stars (excellent)
